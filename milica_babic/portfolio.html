Моји пројекти

Идеје за прва три пројекта нису моје, већ су из сљедећег курса са платформе Udemy, који свима препоручујем: линк. Сви пројекти сем четвртог, шестог и првог су рађени у језику JavaScript, помоћу библиотеке React. Четврти и шести пројекат су рађени у језику Python.

Startup landing page

Овај пројекат је рађен само помоћу HTMLa, CSSa и библиотеке Bootstrap.

Апликација Robofriends

Овај пројекат је урађен користећи библиотеку React, тачније модул Create React App.

Апликација за детекцију лица на сликама

Овај пројекат је урађен користећи библиотеку React, тачније модул Create React App. Главна компонента је поље за детекцију лица са фотографија, чија се адреса унесе у за то предвиђено поље. Апликација онда позива Clarifai API. Слика се прикаже испод поља за унос, са плавим правоугаоницима преко ње, лоцираним око лица. Идеја за пројекат је са курса: линк. Ја сам измијенила то да апликација прикаже до 30 лица из низа враћених података (не само прво једно), као и то да се некад не приказује правоугаоник на сликама гдје нема лица, залутао са слике из претходног уноса.

Игра Понг

Овај пројекат је игра Понг за једног играча, рађена у језику Python, користећи библиотеку PyGame. Постоје два фајла, први је за једног играча (корисника), а други је модел трениран да сам игра користећи алгоритам К-најближих сусједа (импортован из модула scikit-learn).

Конвејева игра живота

Овај пројекат је имплементација Конвејеве Игре живота, чија су правила:линк. Рађена је помоћу библиотеке React (модул Create React App), у језику JavaScript.

Игра Space Invaders

Овај пројекат је игра Space Invaders за једног играча, рађена у језику Python, користећи библиотеку PyGame.

Игра Asteroids

Овај пројекат је игра Asteroids за једног играча, рађена у језику JavaScript.

Игра CryptoCheckers

Овај пројекат је игра рађена у језику JavaScript, за серверски дио се користе Node.js и Express, а за слање порука у чету као и за слање потеза библиотека Socket.io.

Чет апликација

Овај пројекат је једноставна чет апликација рађена у React-у помоћу библиотеке Socket.io, а серверски дио користи Node.js и Express.
